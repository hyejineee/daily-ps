## 📖 학습 가이드

### 난이도 기준
- **★ 기초** (브론즈 ~ 실버 중반) — 패턴 이해, 구현 연습
- **★★ 중급** (실버 중반 ~ 골드 중반) — 응용력, 복합 패턴
- **★★★ 고급** (골드 중반 ~ 플래티넘) — 최적화, 복합 알고리즘

### 추천 학습 순서
```
Ch01 배열 → Ch02 해시 → Ch03 스택/큐 → Ch06 이진탐색
→ Ch04 투포인터 → Ch05 슬라이딩윈도우
→ Ch07 백트래킹 → Ch08 그리디 → Ch09 DP
→ Ch10 그래프
```

### 각 챕터 학습법
1. **개념 정독** — 패턴과 코드 템플릿 이해
2. **기초 10문제** — 막히면 힌트 보되, 스스로 코드 작성
3. **중급 20문제** — 30분 고민 후 막히면 접근법만 확인
4. **고급 30문제** — 풀이 후 반드시 다른 사람 풀이와 비교
5. **단원 모의고사** — 실전처럼 타이머 켜고

- 오답 노트: 틀린 이유를 한 줄로 기록
