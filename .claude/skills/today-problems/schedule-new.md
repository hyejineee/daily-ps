### Day 2 (2/9) - Prefix Sum 활용
**목표**: 부분합을 O(1)에 계산하는 능력

| 문제 | 번호 | 난이도 | 태그 | 비고 |
|------|------|--------|------|------|
| 인간-컴퓨터 상호작용 ⭐ | 16139 | Silver 1 | `prefix_sum` | 2D Prefix Sum |
| Maximum Subarray | 10211 | Silver 4 | `prefix_sum`, `kadane` | Kadane 알고리즘 |
| 구간 합 구하기 4 | 11659 | Silver 3 | `prefix_sum` | 기본 구간 합 |

**학습 포인트**:
- prefix[i] = arr[0] + ... + arr[i-1] 정의
- 구간 [i, j] 합 = prefix[j+1] - prefix[i]
- 2차원 누적 합 확장

## 📊 학습 성과 체크리스트

### 필수 패턴 (5가지)
- [ ] **Monotonic Deque**: O(N) 슬라이딩 윈도우
- [ ] **Prefix Sum**: O(1) 부분합 계산
- [ ] **Two Pointer**: O(N) 투 포인터
- [ ] **Kadane's Algorithm**: 최대 부분합
- [ ] **Parametric Search**: 이분 탐색 응용

### 복합 패턴 (3가지)
- [ ] **BFS + 상태 추가**: 벽 부수기, 0-1 BFS
- [ ] **DP + 공간 최적화**: Rolling Array, O(1) DP
- [ ] **그리디 + 정렬**: 이벤트 정렬, 우선순위 큐

### 실전 능력
- [ ] **시간복잡도 판단**: N 범위 보고 필요 알고리즘 역산
- [ ] **Gold 4 안정권**: 40분 내 해결
